# Приложение «Стоимость телефонного разговора»

**Разработчик:** [Кирилл Кузнецов]  
**Вариант:** [6]  
**Дата:** 03.04.2026

## Описание проекта

WPF-приложение для расчёта стоимости телефонного разговора.  
Пользователь вводит длительность (мин), цену за минуту, выбирает день недели.  
Программа автоматически применяет скидки:

- Если разговор длится **более 30 минут** – минуты с 31-й и далее дешевеют на **30%**.
- Если разговор происходит **в выходной день** (суббота или воскресенье) – итоговая стоимость снижается ещё на **15%**.
